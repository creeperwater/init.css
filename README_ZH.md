# init.css

一个简约的CSS初始化库。

## 简介

该库目前包含了三种情境下的CSS初始化：

- 普通网页（[page.css](src/page.css) page.min.css）
- 仿app网页（app.css app.min.css）
- 游戏网页（game.css game.min.css）

## 属性

### 普通网页

去除了边距，边框宽度包含在总宽度中，去除了各个标签的默认样式，设置了默认字体。
